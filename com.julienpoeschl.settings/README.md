# Personal Unity Settings Package
A [Unity](https://unity.com) 6 package, created using my [Unity-Package-Template](https://github.com/julienpoeschl/Unity-Package-Template), that provides global game settings built on Scriptable Objects.

For more details, read the [documentation](com.julienpoeschl.settings/Documentation/index.md) or [changelog](com.julienpoeschl.settings/CHANGELOG.md).
